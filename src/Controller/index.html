Sheepsmeadow

By Michael Brüggemann

Version: 1.0.0

Introduction

Sheepsmeadow is a simple simlation program, based on the Lecture "Einführung in die Geoinformatik" held by Prof. Dr. Christian Kray at University Münster, Germany. This program aims to give an easy introduction to "grid-based Agent Based Simulations".

Sheepsmeadow let's you choose a gridsize for the "Meadow", define the number of Sheeps and Wolves to place on the Meadow. It let's you also define a probability how likely it is, that agents can create offsprings. At last, you are able to look at distributions of the energy level of all agents, calculated at each timestep.

How to use this program

Sheepsmeadow provides different "Tabs" with options:

Console

The "Console"-Tab let's you change basic simulation properties like the Seed, Step-Delay and Stop/Pause-Criteria. If you want to run the same simulation over and over again, you have to use the same Seed every time and also uncheck the option "Increment Seed on Stop".
The Step-Delay defines how many seconds it takes to perform a step of the simulation. The larger this number gets, the more time it takes to perform a simulation step.

Displays

THe "Displays"-Tab lists all "Displays" provided by Sheepsmeadow. A "Display" is a Window where graphical elements of the simulation (like the "Meadow") are stored. Here you have the option to "show" or "hide" those Displays.

Inspectors

Currently not supported.

Model

The "Model"-Tab provides options to change the "Model Parameters". Detailed information about the "Model Parameters" are provided in Model Parameters.

Model Parameters

Sheepsmeadow provides some parameters for you to edit. Those parameters change the behavior of the simulation (like gridsize, reproducibility of agents, etc.). Sheepsmeadow also provides "read-only" parameters like WolfEnergyDistribution and SheepEnergyDistribution.

GridSize

The GridSize defines the area of the "Meadow". This is a raster of size N x N. The Gridsize is a threshold for how many agents can be placed on the "Meadow" (e.g. with a Gridsize of 5, only 5*5=25 agents can be placed).

ReproductionDelay

The ReproductionDelay defines the number of simulation steps, an agent that has been part of a reproduction event needs to wait, until it can reproduce again.

SheepEnergyDistribution

Using the small "magnifying lens" next to this parameter in the "Model"-Tab, you can find the option "Make Histogram". Click this, and a window with the corresponding histogram will pop up.

SheepFertillityRate

The SheepFertillityRate defines how likely it is, that two sheeps produce an offspring in a reproduction event. "1" means the reproduction will always be sucessful and "0" means the two sheep can't reproduce.

Wolves

Wolves defines, how many wolves should be placed on the "Meadow". You can only places as many wolves, as the the gridsize allows.

WolfEnergyDistribution

Using the small "magnifying lens" next to this parameter in the "Model"-Tab, you can find the option "Make Histogram". Click this, and a window with the corresponding histogram will pop up.

WolfFertilityRate

The WolfFertillityRate defines how likely it is, that two wolves produce an offspring in a reproduction event. "1" means the reproduction will always be sucessful and "0" means the two wolve can't reproduce.

Sheep

Sheep defines, how many sheep should be placed on the "Meadow". You can only places as many sheep, as the the gridsize allows.
