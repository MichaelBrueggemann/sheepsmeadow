Sheepsmeadow

By Michael Brüggemann

Version: 1.0.0

Introduction

Sheepsmeadow is a simple simulation program, based on the Lecture "Einführung in die Geoinformatik" held by Prof. Dr. Christian Kray at University Münster, Germany. This application aims to give an easy introduction to "grid-based Agent Based Simulations".

Sheepsmeadow is a Simulation of Wolves and Sheeps living on a Meadow.
The Meadow is where all action happens. Sheep eat grass, reproduce or get eaten by wolves. Wolves stroll around on the hunt for lonely sheep or try to keep their population alive by producing more wolves.

How to use this application

Sheepsmeadow provides different "Tabs":

Console

The "Console"-Tab let's you change basic simulation properties like the Seed, Step-Delay and Stop/Pause-Criteria.
If you want to run the same simulation over and over again, you have to use the same Seed every time and also uncheck the option Increment Seed on Stop.

The Step-Delay defines how many seconds it takes to perform a step of the simulation. A step is just number that gets increment every time every agent in the simulation has performed their action. The larger Step-Delay gets, the more time it takes to perform a step.

The Stop/Pause-Criteria defines, on what condition the simulation run should be ended/paused. You have the option to either give a specific step (e.g. by giving "10", the simulation run will end/stop at Step 10).

Another option is to set a specific time as a Stop/Pause-Criteria. Time means the "simulation time" not the normal clock-time. This is a bit complicated to explain, so just ignore it.

If the Stop/Pause-Criteria can be applied, depends on the current Simulation Mode. You can change it in the bottom right corner of this window with this dropdown-menu. To make things easier, please only use the mode Steps.

Displays

The "Displays"-Tab lists all "Displays" provided by Sheepsmeadow. A "Display" is a Window where graphical elements of the simulation (like the "Meadow" where you can see all the Sheep and Wolves) are stored. Here you have the option to "show" or "hide" those Displays.

Inspectors

Currently not supported.

Model

The "Model"-Tab provides options to change the "Model Parameters". These change how the simulation behaves.

How to start a simulation

To start a simulation run using Sheepsmeadow, you first need to set the Model Parameters (those are described in detail below).

After you've set all desired parameters, you can start the simulation using the Start Button placed in the bottom left corner of this window. When the simulation run is started, you can pause it any time using the Pause Button .

After pausing, the button changes to the Resume Button . With this button you can increment the simulation step by one, each time you press this button. If you press the Pause Button again, the simulation will continue to run automatically.

To end the simulation run prematurely, you can use the End Button .

Model Parameters

Sheepsmeadow provides some parameters for you to edit. Those parameters change the behavior of the simulation (like gridsize, reproducibility of agents, etc.). Sheepsmeadow also provides "read-only" parameters like WolfEnergyDistribution and SheepEnergyDistribution.

GridSize

The GridSize defines the area of the "Meadow". This is a raster of size N x N. The Gridsize is a threshold for how many agents can be placed on the "Meadow" (e.g. with a Gridsize of 5, only 5*5=25 agents can be placed).

ReproductionDelay

The ReproductionDelay defines the number of simulation steps, an agent that has been part of a reproduction event needs to wait until it can reproduce again.

SheepEnergyDistribution

Using the small "magnifying lens" next to this parameter in the "Model"-Tab, you can find the option Make Histogram.
Click this, and a window with the corresponding histogram will pop up. Please note, that you have to start the simulation, before you can create a histogram. Otherwise the histogram window automatically will be closed.

SheepFertillityRate

The SheepFertillityRate defines how likely it is, that two sheeps produce an offspring in a reproduction event. "1" means the reproduction will always be sucessful and "0" means the two sheep can't reproduce.
!! This parameter can be changed during the simulation run !!

Wolves

Wolves defines, how many wolves should be placed on the "Meadow". You can only place as many wolves, as the the gridsize allows.

WolfEnergyDistribution

Using the small "magnifying lens" next to this parameter in the "Model"-Tab, you can find the option Make Histogram.
Click this, and a window with the corresponding histogram will pop up. Please note, that you have to start the simulation, before you can create a histogram. Otherwise the histogram window automatically will be closed.

WolfFertilityRate

The WolfFertillityRate defines how likely it is, that two wolves produce an offspring in a reproduction event. "1" means the reproduction will always be sucessful and "0" means the two wolve can't reproduce.
!! This parameter can be changed during the simulation run !!

Sheep

Sheep defines, how many sheep should be placed on the "Meadow". You can only place as many sheep, as the the gridsize allows.
